# Integrated-Furniture-Shop-Management-System
A client-based web application for managing furniture retail operations, including product management, orders, payments, deliveries, and customer support.

## 📖 Project Overview

The Integrated Furniture Shop Management System was designed for a real furniture business to improve operational efficiency by automating day-to-day business processes. The application provides separate interfaces for customers and administrators, enabling secure product management, order processing, payment handling, delivery coordination, and customer support within a single platform.

---

## ✨ Key Features

- User registration, authentication, password recovery, and role-based access control.
- Product catalog and inventory management with category, pricing, stock, and image management.
- Shopping cart management, order placement, order confirmation, and order status tracking.
- Secure payment processing, invoice generation, transaction history, and refund management.
- Delivery scheduling, assignment, and real-time delivery status tracking.
- Customer feedback, product reviews, and support ticket management.
- Administrative dashboard for managing users, products, orders, payments, deliveries, and support requests.
- Search and filtering functionality across products, orders, users, and transactions.
- RESTful API architecture with Spring Boot and MySQL.
- Developed using Agile Scrum with Jira throughout the Software Development Life Cycle (SDLC).
